# Test CI Repository

This is a simple test repository for CI/CD pipeline testing.

## Files

- `package.json` - Node.js project configuration
- `index.js` - Simple Node.js application
- `.github/workflows/ci.yml` - GitHub Actions workflow (optional)

## Usage

```bash
npm install
npm test
npm run build
```

## CI/CD Pipeline

This repository is configured to work with SufiCI platform.
